Header

This header uses position: sticky with top: 0. It stays stuck to the top of the screen when scrolling starts, which helps the user keep context. It works well in layouts where the header should remain visible but not always fixed.

Navigation

This navigation bar uses position: relative and is moved slightly to the right using left: 5px. Relative positioning means it keeps its space in the layout, but we can visually shift it. This is useful for fine-tuning placement without breaking the layout flow.

Main Content

The main section uses position: static, which is the default. It stays in the normal document flow. No offsets like top/left apply to static elements. It appears naturally in the center column defined by the grid.

Sidebar

This sidebar uses position: absolute and is shifted down using top: 100px and to the right with right: 10px. It’s taken out of the document flow and placed exactly where specified. This allows it to "float" over other content if needed.

Footer

This footer uses position: fixed and stays pinned to the bottom of the screen with bottom: 0. It’s useful for site-wide footers that should always be accessible, but may overlap content if not spaced carefully.
